# 🎲 Guide de Test - Golden Ludo

Bienvenue dans la version de test de **Golden Ludo** ! Voici les points clés à tester pour nous aider à améliorer le jeu.

## 1. Authentification 🔑
- **Inscription** : Créez un compte avec votre pseudo. Vérifiez que la jauge de force du mot de passe fonctionne.
- **Email** : Vous allez recevoir un email de confirmation. Cliquez sur le lien. Vous devriez voir un message vert : *"Email confirmé avec succès"*.
- **Connexion** : Connectez-vous avec vos identifiants.

## 2. Tableau de bord (Lobby) 📊
- **Bonus** : Vérifiez que vous avez bien **100€** dans votre solde en haut à droite (cadeau de bienvenue !).
- **Historique** : Regardez en bas à droite, vous devriez voir la transaction de bienvenue avec la bonne date.
- **Modes de jeu** : Cliquez sur les différents modes (1vs1, Solo vs IA) pour voir les descriptions.

## 3. Mot de passe oublié ? 🆘
- Si vous voulez tester la récupération, déconnectez-vous, allez sur "Mot de passe oublié", et suivez le lien dans votre email pour définir un nouveau mot de passe.

## 4. Retours d'expérience 💬
- L'interface est-elle fluide ?
- Les couleurs sont-elles lisibles ?
- Avez-vous rencontré des messages d'erreur bizarres ?
